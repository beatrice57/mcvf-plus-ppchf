Syntactic tags

Basic syntactic tags

Syntactic tags encode various pieces of information about a phrasal constituent, notably its syntactic category and grammatical or thematic function and also various other syntactic, discourse, and metalinguistic features.

Certain basic syntactic categories have variants ending in X (for instance, NP and NX). These variants are used in connection with shared premodifiers in conjunction contexts and are intended to be agnostic about whether the category in question is a lexical or a phrasal projection. For details, see Conjunction.

Function tags

The following function tags can be appended to the basic syntactic tags above (except for PPs).

-ADT adjunct
-ADV adverbial
-CAR clause-adjoined
-COM bare NP complement
-DIR directional
-LOC locative
-MSR measure
-OB1 direct object
-OB2 indirect object
-PART partitive
-PRD predicate
-RFL inherent reflexive
-SBJ subject
-SPR secondary predicate
-TMP temporal
-VOC vocative

Extended tags

The following extended tags can be appended to basic tags or to function tags.

-LFD dislocated constituent (generally left-dislocated, but also used for right dislocation)
-PRN parenthetical or appositive
-RSP resumptive constituent
-SPE direct speech (only on CP, IP)
-XXX unknown function

"Meta" tags

CODE material that is not part of the text proper (text markup, page numbers, editorial comments, and the like)
LATIN passage in Latin
LS list marker
REF references in the original text to sources of quotations

*proimp* empty expletive subject

empty terminal node (when immediately dominated by POS label)

Indices

Numeric indices are used to coindex:
  • antecedents with their traces
  • expletives (overt or empty) with their associates (whether clauses or noun phrases). Indices on the expletive and associate clause are set off by hyphen and equal sign, respectively.
  • gapped clauses with the full clauses that license them (see Gapping, Right node raising). Indices on the full and gapped clause are set off by hyphen and equal sign, respectively.
