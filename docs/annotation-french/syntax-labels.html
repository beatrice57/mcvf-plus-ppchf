Function tags

The following function tags can be appended to the basic syntactic tags above (except for PPs). Function tags that are (almost) entirely restricted to particular basic categories are listed under the relevant basic category.

-ADT adjunct
-ADV adverbial
-CAR clause-adjoined (relative)
-DIR directional
-EXL exclamative
-LOC locative
-OPT optative
-PRD predicate
-SPR secondary predicate
-TMP temporal

Extended tags

The following extended tags can be appended to basic tags or to function tags.

-LFD dislocated constituent (generally left-dislocated, but also used for right dislocation)
-PRN parenthetical or appositive
-RSP resumptive constituent
-SPE direct speech (only on CP, IP)
-XXX unknown function

Empty categories

0 empty operator
*arb* empty subject in causative constructions
*con* subject elided under conjunction (default)
*proimp* empty expletive subject
*pro* unambiguous "small pro"
*ICH* abbreviation mnemonic for "insert constituent here"; trace of extraposition, scrambling, or other movement that does not fit neatly into the A/A' dichotomy
*T* trace of wh- movement
* trace of A-movement

Indices

"-#" (a hyphen followed by a numeric index) is used to coindex antecedents and their traces, as well as expletives (overt or empty) that are associated with a clause or noun phrase.

"=#" (an equals sign followed by a numeric index) is used to coindex gapped clauses with full clauses. See Gapping, Right-node raising.
